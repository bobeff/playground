# От цифри към числа

Напишете функция `from-digits`, която приема последователност от цифри и връща
числото, съставено от тези цифри. Ако последователността е празна, функцията
трябва да върне 0:

    (from-digits [4 2]) ; 42
    (from-digits [])    ; 0

За целите на задачата "цифра" е цяло число от 0 до 9.
